# GoSymGym

[![Go Version](https://img.shields.io/badge/Go-1.22%2B-00ADD8?logo=go)](https://golang.org)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![CI](https://github.com/Gitubrr/GoSymGym/actions/workflows/ci.yml/badge.svg)](https://github.com/Gitubrr/GoSymGym/actions/workflows/ci.yml)

**GoSymGym** is a CLI tool for getting detailed information about GitHub repositories directly from the terminal. It displays stars, forks, issue count, language, creation and update dates, and a link to the repository.

---

## Installation

### Method 1: Installing with `go install`

```bash
go install github.com/Gitubrr/GoSymGym/cmd/GoSymGym@latest
